# ZyLabs AI Research Copilot Frontend

React Single Page Application built on Vite, React 18, Tailwind CSS, Zustand, and React Query, visualizing real-time sales briefings pipeline.

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

## Folder Structure

- `src/api/`: Endpoint handlers.
- `src/components/`: Modular widgets like create session form, stepper progress indicators, and chatbot panel.
- `src/hooks/`: WebSocket events streaming listeners and API query connectors.
- `src/store/`: Zustand state definitions.
- `src/types/`: TypeScript definitions.
- `tests/`: Vitest snapshots and layout interaction tests.
